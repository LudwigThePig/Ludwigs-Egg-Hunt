# Some Socket Bop 'Em Block 'Em!

## Wireframe
1. Server serves up static and index.html
2. User picks a name
  - Emits a post request
  - Mongo stores name, score, and isOnline
3. Socket On Connect
  - Put updates isOnline
4. Play the game!
  


## Technology
1. Javascript
2. Node/Express
3. Socket.io
4. Mongo
